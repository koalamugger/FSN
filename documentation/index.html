Customizing Navigation

<div class="sidebar-navigation">
    <a href="index.html"  class="sidebar-icon sidebar-1"></a>
    <a href="type.html"   class="sidebar-icon sidebar-2"></a>

    <div class="clear"></div>

    <a href="folio.html"   class="sidebar-icon sidebar-3"></a>
    <a href="#" class="close-sidebar sidebar-icon sidebar-4"></a>
    <a href="jquery.html"  class="sidebar-icon sidebar-5"></a>

    <div class="clear"></div>

    <a href="gallery.html"   class="sidebar-icon sidebar-6"></a>
    <a href="blog.html"    class="sidebar-icon sidebar-7"></a>
</div>

This is the page navigation, it can be found in the sidebar class. To edit the navigation simply change the hrefs locations. To change the icons, check the design folder, we've included the PSD to the navigation to edit it. 

You can use two navigations, further editing requires changing the menus-background.css or any of the css's you use in the style folder.

The items in the sidebar are styled as hexagone, so there are 2 navigation items per row and a close button in the middle of the second row. That icon has the class close-sidebar.

 

Customizing Updates

<div class="page-notifications warning-notification">
    <h4>Notification Title</h4>
    <p>Sidebar updated to have icons and text descriptions!</p>
</div>

<div class="page-notifications information-notification">
    <h4>Notification Title</h4>
    <p>Sidebar updated to have icons and text descriptions!</p>
</div>


<div class="page-notifications alert-notification">
    <h4>Notification Title</h4>
    <p>Sidebar updated to have icons and text descriptions!</p>
</div>


<div class="page-notifications confirmation-notification">
    <h4>Notification Title</h4>
    <p>Sidebar updated to have icons and text descriptions!</p>
</div>

The sidebar updates. You can change the text with whatever you wish and change the style by editing the warning-notification information-notification alert-notification and confirmation-notification. Each one of these will change the color of your update.

 

Stylesheets

The stylesheets that change the colors of the sidebar elements are as follows. The default background image sidebar is currently menus-background.css. 
 
You can switch between
 
menus-background.css - background image theme 
menus-variations.css - rainbow theme
menus-blue.css - blue theme
menus-green.css - green theme
menus-orange - orange theme
menus-magenta - magenta theme
menus-red - red theme
menus-white - white theme
